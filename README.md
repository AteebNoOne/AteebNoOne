## About

I'm a full-stack developer and technology lead with a focus on building production-grade systems across the stack — hardware integration, backend infrastructure, and applied AI. As the lead of **Teebot** (Systems, Labs, Consulting, and Studio), I work across disciplines to take products from architecture to deployment.

- 🦀 Building **tinyxml2-rs**, a memory-safe Rust replacement for the TinyXML2 C++ API
- 🤖 Developing **Teebot Stream**, a real-time AI speech translation system, and automated video agents built on Gemini/Groq orchestration
- 🐧 Contributing to Linux kernel development (staging drivers) and auditing production Docker deployments for security
- 🧭 Mentoring engineers on AI application architecture and system design
